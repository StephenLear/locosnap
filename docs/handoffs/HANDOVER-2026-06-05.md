## Lessons Learned & Gotchas
- **The haul/gacha format is a PL winner and a DE loser** — opposite of the channel's normal DE>PL retention. Don't run a DE haul.
- **A modernisation sub-variant does NOT always inherit the base family's design speed** (EN57AKM 120 vs EN57 110). Verify the specific variant.
- **The "5m cache write" band is an Anthropic accounting artifact, not churn** — the write-amortization figure is the real health signal. Do not re-investigate.
- **RevenueCat never pays you** — Apple + Google disburse directly (separate schedules); RC is metered middleware that charges you.
- **Chrome MCP pairing drops on logout** — `list_connected_browsers` returns `[]`; the user must re-open Chrome / click Connect. Have a CSV/screenshot fallback ready.
- **Supabase migrations are NOT auto-applied by Render** — committing migration 020 is inert until pasted into the SQL editor. Prod schema is still at 019.
