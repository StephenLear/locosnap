## What Worked
- APT-P scan through LocoSnap returning LEGENDARY with "3 built, none survive" — perfect hook content
- ImprovMX + Resend split (sending vs receiving) — clean solution for email
- EAS `autoIncrement: true` — build number handled automatically
- 39/39 frontend tests passing cleanly before build
- Formula 1 viral hook: "Can you name this train?" creates open loop that stops scroll

## What Didn't Work (and Fixes)
- **ImprovMX SPF missing** — gave DNS records without root @ SPF for ImprovMX. Fixed by adding `v=spf1 include:spf.improvmx.com ~all` at `@` → saved to email-send-guard skill
- **EAS not in PATH for background tasks** — `eas build` failed with "command not found". Fixed using full path `/usr/local/bin/eas` with `PATH="/usr/local/bin:$PATH"` → saved to pre-build-git-verification skill
- **Wrong logo in email** — iTunes API returned wrong image. Fixed using GitHub raw URL → saved to email-send-guard skill
- **Email sent without approval** — sent test email before user approved. Fixed → saved to email-send-guard skill
- **Endcard logo spacing took 4 iterations** — no systematic approach to y-coordinate spacing. Fixed rule: 200px logo, 80px minimum gap → saved to video-editing skill
- **Pro SQL "no rows returned"** — JOIN syntax issue. Workaround: SELECT UUID first, then UPDATE by ID directly
- **German release notes not offered** — user had to ask. Fixed → new app-store-release-notes skill

---

## Lessons Learned & Gotchas
- Always provide English + German release notes together — don't make user ask
- EAS CLI in background bash: always use `/usr/local/bin/eas` with explicit PATH
- Supabase UPDATE returning "no rows" → first run SELECT to confirm row exists, then UPDATE by UUID
- ImprovMX requires SPF record at root `@` domain IN ADDITION to Resend's SPF at `send` subdomain
- Android session fix is Android-specific — do NOT include in iOS release notes
- Most testers are in guest mode — Pro SQL grant will only work once they create accounts and sign in
- TikTok drops at 0:01 without audio — next video should have music
