## Lessons Learned & Gotchas
- Read ARCHITECTURE.md at the very start of every session — it is the only reliable state (rule added to engineering-session-standards)
- Android tester email has a fixed template — never rewrite from scratch, only update APK link and bullets (rule added to engineering-session-standards)
- Always extract a frame from the latest performing ad before building a new one — match font/colour/size exactly (rule added to video-editing skill)
- Preview the result card frame before concatenating — confirm train class matches hook footage (rule added to video-editing skill)
- zoompan is too slow for clips > 5s — avoid it (rule added to video-editing skill)
