## Lessons Learned & Gotchas
- Always clarify "TestFlight or App Store?" before giving submission instructions — never assume
- Never suggest changing an App Store submission field from its previously set value without explicitly flagging it
- Don't ask for tester emails/info if user has indicated they don't need to add testers
- Resend "Verify DNS Records" button is near the top of the DNS records section, not at the bottom of the page
- Render auto-deploys on push to main — no manual steps needed
