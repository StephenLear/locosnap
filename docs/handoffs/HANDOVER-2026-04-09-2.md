## Lessons Learned & Gotchas

- When a user says a video's format is wrong, always rebuild from source — never assume a component swap fixes it
- Never write train disambiguation rules from text research alone — always extract and review actual footage first
- The ICE L (Talgo) is loco-hauled by a Vectron BR 193, not a self-propelled EMU. The height step-down between loco and coaches is the key visual identifier
- TikTok viewer geography and follower geography are very different metrics — followers: DE 15.1%, UK 14.9%, PL 10.3% but viewers: PL 67.4%, DE 8.4%, UK 3.8%. The viewer stat is what matters for algorithm targeting
- Instagram handle is @locosnap_app (not @locosnap which is squatted)
- DB on-time stat for captions: only 62.1% of long-distance trains arrived on time in March 2026 (DB's own data)
