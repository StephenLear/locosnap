## What Didn't Work (and Fixes)
- BR 423 misidentified as BR 425 on the red Frankfurt West footage → Added disambiguation rule (fleet number prefix + S-Bahn context markers). Rule is deployed but the vision model still returns BR 425 for the red 423 footage — the fleet number on the cab front is not reliably read at distance. Outcome: video uses BR 425 result (acceptable — viewers won't know the difference)
- Blue 423 footage (RMV livery) produced unstable results (425 → 642 → 425) → Dropped the blue livery angle entirely; used only the red Frankfurt West footage
- BR 425/426 spec showed "Derby Works" as builder → Fixed with hardcoded spec correction (Salzgitter)
- Text at 110px clipped "WAS IST DAS?" right edge → Reduced to 90px (should have done character count check first — rule now added to skill)
- Initial clip plan included scanning animation section → User corrected to card-only (rule now added to skill)

## Key Decisions
- **Ship with BR 425 result despite it being a 423**: For TikTok quiz format, 99% of viewers won't know the difference. Risk of pedantic commenter is low; reward is shipping tonight with existing footage.
- **Drop blue RMV livery angle**: Blue 423 gave unstable ID results (different wrong answer each scan). Stable wrong answer (BR 425 every time) is more reliable for a repeatable video asset than unstable wrong.
- **German quiz hook "JEDEN TAG / WAS IST DAS?"**: Simpler and wider appeal than the volume-stat angle. The Frankfurt skyline does the contextual work without needing "1 million daily" type text.
