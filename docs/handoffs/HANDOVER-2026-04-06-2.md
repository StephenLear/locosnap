## Key Decisions

- **Replace `<Redirect>` with useEffect everywhere** — Not just this instance. `<Redirect>` is unsafe on Android 16 due to Zustand's synchronous `useSyncExternalStore` causing parent re-renders on every navigation event. Any future navigation-gating should use useEffect + router.replace().
- **Skip FCM on Android remains in place** — v1.0.16 inherits all previous fixes including FCM skip. Push notifications remain non-functional on Android until that is revisited.
- **Class 14 rule uses fleet number range as primary signal** — D9500-D9555 is unique and definitive. Size and context are secondary signals.
- **ICE 1 video in German** — Matched Frankfurt formula (recognisable location / train + local language). German audience is the strongest TikTok segment (Frankfurt: 50.7%).

## Lessons Learned & Gotchas

- **`<Redirect>` is a React component, not a one-shot redirect** — It runs its internal useEffect on every mount. On Android 16, it mounts on every navigation event. Never use it for conditional routing in `_layout.tsx`.
- **EAS build profile "preview" = internal distribution = APK** — Must specify `--profile preview` not `--profile production` when building APK for testers.
- **nvm must be sourced in every bash session** — `eas` is not in PATH by default. Use: `export NVM_DIR="$HOME/.nvm" && source "$NVM_DIR/nvm.sh" && nvm use 20`.
- **German captions need umlaut verification** — "fahrt" was corrected to "fährt" after the fact. Always verify before posting.
- **APK build history must be updated the same session the APK is sent** — Was a pending task at handover start this session; confirms the rule is worth enforcing strictly.
- **End screen asset location is `docs/assets/locosnap_end_screen.mp4`** — If this file doesn't match what's been posted, extract fresh from `Desktop/EU07/locosnap_eu07_v4.mp4` at 7.5s.
