## Lessons Learned & Gotchas
- **Vercel migrated its apex IP:** old `76.76.21.21` / `cname.vercel-dns.com` still work but are flagged; new apex A = `216.198.79.1`, www CNAME is now PER-PROJECT (`703acd8d3064cca9.vercel-dns-017.com`). The real "Invalid Configuration" culprit was a stale **AAAA** record (`2600:1901:0:84ef::`) — IPv6 clients hit it first and fail.
- **ČD 242 zone has NO MX/SPF** — `@unsunghistories.com` mail isn't routing (separate ImprovMX task if wanted; renewing Hostinger hosting would NOT fix it).
- **KNOWN_SPECS exact-match keying trap (again):** lookup is `WIKIDATA_CORRECTIONS[class.toLowerCase().trim()]` — must key the FULL vision class string. Added all diacritic/non-diacritic + 73E variants for the 242.
