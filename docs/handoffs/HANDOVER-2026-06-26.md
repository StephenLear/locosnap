## Lessons Learned & Gotchas
- **When a third-party API fails universally, check model/endpoint deprecation AND the usage/spend signal BEFORE billing or transient-infra causes.** $0 image spend = nothing is generating. Repeated "transient" diagnoses on the same feature = a systemic root cause being missed.
- **For look-alike families (VR Dv12/Dr16/Dr19), lead the vision rule with the FLEET NUMBER, not the silhouette.** A visual-only rule silently fails on a unit that doesn't match the assumed styling.
- **Google Maps Android keys can't be kept secret** — they ship in the APK. GitGuardian will flag them forever; the security model is application restriction (package + SHA-1) + API restriction, not secrecy. Verify restrictions live in GCP before dismissing such an alert.
- **gpt-image-1 ≠ dall-e-3 drop-in** — different `size`/`quality` enums, no `style`, returns base64 not a URL. Each mismatch is its own 400.
