## Lessons Learned & Gotchas

- **Dr18 is Fenniarail, NOT VR.** Our Dr-series vision prompt frames all Dr-classes as VR-owned, which excluded the correct answer entirely. General lesson: a class designation with a national prefix doesn't imply a specific operator. Six Dr18 units exist (Dr18 101–106), all CZ Loko 2015–2020, dark-green + yellow Fenniarail livery, Co'Co' six-axle hood unit. The next backend session has everything needed to write the rule cold.
- **Dv12 livery: red-with-light-grey, not orange-white.** The old memory note's "orange/white VR livery" for Dv12 is wrong. Historic scheme is red-with-light-grey, later green-white, now modern white-with-green-stripe. Always verify a specific livery claim against source photography before hardcoding.
- **Don't pattern-match from a single data point.** "Class 37 IG landed 71% German" doesn't mean "all UK EN content will land German on IG" — Class 91 IG was 37.3% UK. Read the full dataset before recommending a strategy shift.
- **Project status memory drifts from backlog completed section.** The `project_status.md` "v1.0.20 bundle" list included items that were already DONE per `frontend_backlog.md`. Sync these two files periodically; don't trust either in isolation.
- **App Store Connect "Proceeds by Territory" = 100% Germany is the first hard monetisation number.** Germany is the only paying market. The UK paywall hit from 2026-04-16 still has not converted 30h later. The content funnel is working (German downloads +483%), but conversion and retention are the real bottlenecks.
- **Class 91 compounded strongly at 48h: +66.9% views, UK share grew 22.2% → 35.3%, full-watch 16.7% → 20.2%.** Worth watching whether Class 37 and Class 345 follow the same compounding curve.
- **Re-used Class 37 video's end-screen as beat 5** for Class 345 — saved rebuilding it. The end screen is generic enough ("Free on App Store / Coming soon to Android") to be reusable across videos.
