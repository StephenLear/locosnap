## Lessons Learned & Gotchas
- **TikTok post dates: decode the video ID, don't trust OCR.** `id >> 32` = unix timestamp.
- **Anthropic API Console cost = LocoSnap app scan cost.** Claude Code is a separate normal subscription, NEVER billed via the API — report cost figures straight, no dev-session caveats. (`feedback_anthropic_cost_no_dev_caveat`)
- **Sentry "Regressed in `<release>`" is usually coincidental** — Sentry tags the current release on recurrence; check "First seen" for the true origin. (The `REACT-NATIVE-B` 500 showed `682cd7e` but first appeared 3 months earlier.)
- **Upstream-error signature**: body `{"type":"error","error":{"type":"api_error",…},"request_id":"req_…"}` = Anthropic's own error (the `req_…` is theirs) = not our bug. Now documented as a triage playbook in `ARCHITECTURE.md`.
- **Atlas reads best at Europe-wide zoom** — frame wide for reveals, not zoomed-in (data still thin per-region).
- **HST**: no clean raw footage in the bank — Class 91 is the UK high-speed fallback.
