## What Worked
- Freightliner footage (moving train) as 1-second hook — correct instinct, the motion stops the scroll
- Cycling 4 blueprint styles in sequence — shows variety of the feature clearly
- ffmpeg frame extraction to verify each section before concat — caught icon size issue before delivery

## What Didn't Work (and Fixes)
- Scan-first approach on Instagram ad — 26 views, algorithm killed it → rule now in ARCHITECTURE.md
- Blueprint v1/v2 built without logo on end screen → v3 rebuilt with icon, rule saved to video-editing skill and ARCHITECTURE.md
- Icon too small (180px) on first logo attempt → rebuilt at 280px
- Searched wrong files when asked about "the list" from previous session → check-handovers-first rule added to CLAUDE.md

## Key Decisions
- **Freightliner footage over DB Cargo idling**: Motion beats static even if the static shot is cleaner — Freightliner approaching at Crewe is the better hook
- **Use all 4 blueprint styles in one video**: Demonstrates feature variety in a single post — more persuasive than showing one style
- **EU07A rule pushed immediately**: Two independent commenters = validated gap. Same-session fix maintains credibility
- **Do not commit eas.json**: Change contains local absolute path (`/Users/StephenLear/locosnap-play-store-key.json`) that will not exist in EAS Build environment — would silently fail Android auto-submit

## Lessons Learned & Gotchas
- When user references a list or plan from a previous session, check docs/handoffs/ newest-first — not project content files
- Never invoke apple-review-response for user-written reviews — only Apple crash rejections
- Always check video production standards (ARCHITECTURE.md Section 21) before starting any video build
- eas.json service account key path must be an EAS secret, not a local absolute path — fix before attempting Android auto-submit again
- EU07A correction came from two separate commenters independently — technically engaged Polish audience behaves like the German audience: fast to correct, worth engaging carefully
