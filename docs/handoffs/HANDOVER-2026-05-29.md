## Lessons Learned & Gotchas
- **Don't read raw early view count as success** — 554 views means distribution is moving, not that retention is good. The metric that matters for this ad is full-watch % vs the 20.4% ceiling.
- **20.5% vs 20.4% is noise, not a lift** — a 0.1pp gap at 554 views is statistically flat. Early retention often starts slightly high before the broader audience pulls it toward the true number; needs ~1k+ completed views to settle.
- **"Tied with the ceiling" is still a decent outcome here** — this ad does more work (two locos, quiz hook, correction comments) than the 2026-05-26 single-loco €1 ad. Matching retention while generating more comment activity is a net positive.
- A commenter's `start` / operator name can read as a typo — when a fan corrects a scan, take the literal reading seriously before assuming error.
