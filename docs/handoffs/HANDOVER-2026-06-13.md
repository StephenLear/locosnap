## Lessons Learned & Gotchas
- **The free-scan tier is 3 lifetime, no refresh** (client-enforced; backend ceiling 6). "Only 3 scans" complaints are now CORRECT — handle as a reframe, never tell the user to update expecting 6. This is the single fact most likely to trip a future session (the old "6" framing lingered in the memory index for days).
- **Polish past-tense verbs are gendered** — when a commenter's pronoun is unknown, phrase replies impersonally (e.g. "taniej niż się wydaje", not "niż myślałeś") to avoid assuming gender, per the they/them-default rule.
- **The tier-debate ad shape is cross-border by nature** — pan-European icons (ICE 1) and pan-PL legends (EN57) leak ~13-15% to the mirror market. Don't read a 68-75% home-lock as a targeting miss on these; it's the format. Pure-market hero ads still lock 87%+.
- **Backend backlog memory decays on "awaiting push" claims** — always `git log origin/main..HEAD` before treating any "NOT YET PUSHED" note as live. They were all stale this session.
