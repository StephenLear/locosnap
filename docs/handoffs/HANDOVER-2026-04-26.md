## Lessons Learned & Gotchas

- **Sonnet → Haiku migration requires a three-step audit per service** — explicit temperature, substring-extraction parser, and same-input-twice determinism test. Yesterday's `da2e16d` skipped all three; today's `27cf25a` retroactively applied them all. Documented in CHANGELOG and ARCHITECTURE so the next migration doesn't repeat.
- **When adding a new class to the LocoSnap disambiguation system, all FOUR services need updates** — vision.ts (positive anchor), trainSpecs.ts (WIKIDATA_CORRECTIONS), trainFacts.ts (framing override), AND rarity.ts (tier anchor + descriptor lock). Three-of-four is incomplete and will leak the wrong narrative through the rarity short-summary descriptor. The BR 155 session proved this — `90c78ca` covered three services and `faed661` was a same-day cleanup commit because rarity.ts was the missing fourth.
- **Public commenters' factual corrections are the highest-quality test signal we have for the backend.** S.1702's "die 430 fährt nur 140kmh keine 160" comment drove a same-day backend fix that prevents every future BR 430 scan from showing wrong specs. Public correction = public proof point that the app respects expertise. Same pattern as Steph's three-strikes case yesterday and Oula's Sm2 trailer-numbering signal the day before.
- **The IG retention cliff at 0:02–0:03 is universal across this account.** Confirmed across 10+ logged ads via the internal data synthesis agent. This is structural, not hook-specific. A 4s cut where the payoff lands at 0:03 mathematically pulls the entire ad inside the engagement window. The 4s IG variant we shipped tonight tests this hypothesis directly.
- **Agent prompts that need web research must explicitly name the tools.** First external research agent claimed no web tools; second agent (with explicit "You have WebSearch and WebFetch tools. Use them.") ran cleanly. Inferred tool capability is not the same as instructed tool capability.
- **The Steve backup external drive at `/Volumes/Steve backup/Locosnap/`** contains all the old source folders that were cleared from Desktop (BR101, BR111, BR120, BR143, BR151, BR218, BR232, plus untouched ones — Class 345, Class 55 Deltic, EN57, EU07, HST, Frankfurt S Bahn, ICE 1, ICE L Talgo).
