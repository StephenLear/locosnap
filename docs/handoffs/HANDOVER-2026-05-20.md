## Lessons Learned & Gotchas

- **Cross-worktree handover state matters at session start.** A clean `ls docs/handoffs/` on main can hide 1-3 worktree-only handovers. Future sessions should do `git log --all --diff-filter=A -- docs/handoffs/HANDOVER-<recent dates>` to catch worktree-only artifacts.
- **TikTok WebFetch returns only the static loading title.** Confirmed twice this session despite prior-session reports of the alt-attribute trick working. Don't waste retries — fall back to user-paste or memory lookup immediately.
- **The "drop-at-0:02" retention pattern is universal across the channel.** BR 101 v2, BR 420, ÖBB 1144 all show the same drop. Visible loco in frame 0 is the cheapest mitigation; the V 200 hero shot with the V-stripe filling the centre at t=0 was designed against this pattern.
- **Visual-paradox hook (BR 101 v2 "Sieht aus wie 1962, ist aus 1996" formula = 21% full-watch) is the current winning shape on this channel.** Comparison-question hook (BR 420 "Alt oder Neu?", ÖBB 1144 "Alpenstaubsauger?") is consistently under-performing at 8.4-14.1% full-watch.
- **Cross-watched creators tell you adjacent niches.** AutoScout24 + BMW appearing as "Creators your viewers also watched" surfaces a car-enthusiast adjacency in the train audience. Not necessarily actionable today but worth banking — content collaborations or hashtag bridges could exploit it later.
- **One ÖBB 1144 ad got 31.3% Austrian viewers + 14.1% full-watch; the other got 2.5% + 8.4%.** Same ad, different algorithm distribution. Local-audience pull is a 5-7pp full-watch lever. Implication: ad targeting / first-hour engagement determines algorithm push country.
- **First brand-search signal "LocoSnap" appeared in TikTok search queries at 0.4%** (lower-bound display threshold). Real volume is single-digits/week on a 5.5K-view base. Not "brand has broken out" — just first measurable signal that the name is functioning as a search destination.
- **Memory file `footage_source_trainspotting101.md` had already flagged the BR 111 video `7599315127300623638`** as "First usable footage for the BR 111 ad candidate" on 2026-05-17. The user's link sent today was the same URL — closing a previously-staged loop. Memory hit replaced a 5-minute WebFetch + DM round-trip.
- **V 200 / BR 221 unit-number range:** V 200.0 = 001-086 (Krauss-Maffei + MaK, 1953-59), V 200.1 = 101-150 (later BR 221, 1962-65). V 200 122 is in the 101-150 range = V 200.1 = 1962-65 build window. Easy gotcha if you say "1953" on a 122-numbered unit.
