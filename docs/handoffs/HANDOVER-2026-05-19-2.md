## Lessons Learned & Gotchas
- **Arial Black width ratio is ~0.7 of fontsize per char, not 0.6.** The existing `feedback_ad_text_width_rule.md` chart (110px → 9-10 chars max) holds, but mental math should use 0.7 not 0.6 when extrapolating to other sizes.
- **`ad_assets/endcards/` is gone.** Memory pointer `ad_assets_endcards.md` is stale. Regeneration script in that file still works (verified this session). EN + PL endcards need rebuilding next time they're needed. Consider committing the regenerated endcards somewhere persistent.
- **"Alpenstaubsauger" is railfan slang applied loosely.** Originally the 1822, more recently the 1216, but used colloquially for any high-power Alpine ÖBB loco. Don't take captions at face value — frame-check the loco itself.
- **TikTok profile pages return only static HTML in headless fetches** — per-video metadata loads via XHR after mount. To find a specific clip, ask the user for the URL, then `WebFetch` the video page directly (alt-text of the still image carries the caption).
- **yt-dlp impersonation warning is benign** — clip downloaded fine despite the "no impersonate target available" warning. Don't waste tool calls trying to fix it.
- **The 1144 vs 1216 distinction is interesting for future content** — 1144 = Austrian-built (SGP/Elin), 1970s/80s design, single-system AC, universal duty; 1216 = Siemens Eurosprinter, modern 4-voltage multi-system, world speed record holder (357 km/h). Both run on the same tracks but they're completely different generations.
